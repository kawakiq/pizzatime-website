# PizzaTime - Сайт доставки пиццы
Одностраничный сайт доставки пиццы с:
- Интерактивным меню с фильтрацией
- Корзиной покупок
- Оформлением заказа с валидацией
- Адаптивным дизайном на Bootstrap 5

## Технологии
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript
- localStorage

## Как запустить
Просто откройте `index.html` в браузере
